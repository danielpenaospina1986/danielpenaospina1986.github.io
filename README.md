# Portfolio — Daniel Peña Ospina

Sitio estático (HTML/CSS/JS, sin dependencias) listo para GitHub Pages.
Estética: interfaz tipo apple.com + material Liquid Glass.

## Cómo publicarlo en GitHub Pages (10 minutos)

1. Crea una cuenta en github.com si no la tienes (usa tu email profesional).
2. Crea un repositorio nuevo llamado exactamente: `TUUSUARIO.github.io`
   (ej. si tu usuario es `danielpenaospina`, el repo se llama `danielpenaospina.github.io`).
   Márcalo como **Public**.
3. En el repo, usa "uploading an existing file" y arrastra TODO el contenido
   de esta carpeta (index.html, carpetas assets/ y fonts/, este README).
   Importante: index.html debe quedar en la raíz del repo.
4. Commit changes.
5. Espera 1-2 minutos. Tu portafolio estará en: https://TUUSUARIO.github.io

## Para actualizarlo después
Reemplaza los archivos en el repo (Add file → Upload files) y haz commit.

## Notas
- El botón "Download CV" descarga assets/Daniel_Pena_Ospina_CV.pdf.
  Para actualizar el CV, reemplaza ese archivo manteniendo el nombre.
- Para agregar o editar proyectos, modifica el array `projects` dentro de index.html.
